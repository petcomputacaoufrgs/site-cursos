  • O Que é o Git?

    Se eu fosse resumir a resposta, ela seria: Git é uma ferramenta distribuída de controle de versionamento de software. As alterações de uma versão do software são chamadas "commit". Os arquivos que contêm as versões são chamadas "repositório".

    Curso: Uma Introdução ao Git

  • O Que é LaTeX?

    O LaTeX é uma plataforma Open Source voltada a criação de artigos científicos  -  especialmente nas áreas da matemática e ciências exatas - com o paradigma What You See Is What You Mean, que se utiliza do typesetting (ou distribuição) TeX.

    Curso: Introdução ao Latex
